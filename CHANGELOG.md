# Changelog

All notable changes to this project will be documented in this file.

## [0.2.0] - 2026-02-09

### Added
- **Modern UI**: Completely redesigned the interface to resemble `btop`, featuring a sleek dashboard layout.
- **System Statistics**: Added real-time monitoring of CPU, Memory, Disk, and Network usage.
- **Visual File Selector**: Implemented a visual popup menu for selecting video files.
- **Manual Input**: Added support for manually entering file paths or dragging and dropping files.
- **Theming**: Improved color scheme with gradients and better visual hierarchy.

### Changed
- **Rendering Engine**: Refactored to use `ratatui` for more robust terminal UI rendering.
- **Performance**: Optimized the rendering loop for smoother playback and interaction.

## [0.1.0] - 2026-02-05

### Added
- Initial release of Vodeo2ASCII.
- Basic video to ASCII/Pixel art conversion.
- Support for dragging and dropping files.
- Two rendering modes: Pixel Art and ASCII Art.
